# Archaeological Ambiguity

## Principle

Do not collapse authorship questions into a single clean answer.

The world should preserve uncertainty about whether visible art and sacred imagery depict:

- the original builders
- later inheritors
- later religious claimants
- symbolic figures rather than historical people

## Example

Murals showing bird-headed figures in temples should not be treated as straightforward evidence that the original builders literally looked like that.

They may instead represent:

- late symbolic reinterpretation
- mythologized engineers
- priestly claimants to older authority
- distorted cultural memory

## Design Use

Art should function as:

- partial documentation
- symbolic memory
- political claim
- reverent misunderstanding

This ambiguity strengthens the distinction between visible culture and hidden infrastructure.
