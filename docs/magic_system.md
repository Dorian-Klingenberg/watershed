# Magic System

## Overview
Magic is not just a combat feature. It is a philosophical and systemic layer about how power is obtained.

## Light Path
The light path represents:
- study
- meditation
- disciplined perception
- alignment with the deep structure of reality
- slow and difficult mastery

This path should feel:
- precise
- earned
- stable
- clarifying
- transformative

Potential gameplay effects:
- better system perception
- safer interventions
- richer diagnostics
- access to hidden patterns or long-range dependencies

## Dark Path
The dark path represents:
- learning invocation methods
- speaking the language of spirits
- obtaining leverage from external entities
- fast access to power without proportional understanding

The spirits are not automatically evil. They are external intelligences with their own motives. They help when they care, but their incentives need not align with human wellbeing.

This path should feel:
- tempting
- fast
- effective
- unstable
- strategically dangerous

Potential gameplay effects:
- quick intervention power
- brute-force changes
- information or leverage that bypasses ordinary study
- side effects, distortions, dependencies, or manipulation risk

## End-State Insight
Complete mastery of the light path may produce a kind of detachment from worldly concerns. Meanwhile, spirits may care about the world intensely, but from motives that are not benevolent.

This tension should remain intellectually interesting rather than collapsing into a simple moral binary.

## Design Rule
Magic should reinforce the game’s main theme:

> power acquired without understanding carries hidden costs.
