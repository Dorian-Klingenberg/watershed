This bundle captures the current design state for the cozy ancient-infrastructure tutorial area of the larger systems-first water/infrastructure game.

Current direction: **no Thra, no Dark Crystal, no Ulgra**. Use the user's original uploaded map and Ancient Water mood board, plus the pre-Thra generated concept images.

Suggested use:

1. Put this folder under `docs/concepts/grannys-house-trials/` in the project repo.
2. Give Codex or another coding/design LLM the README plus `docs/00_project_brief.md` first.
3. Use the reference images as visual anchors only; use the markdown docs as source of truth.
