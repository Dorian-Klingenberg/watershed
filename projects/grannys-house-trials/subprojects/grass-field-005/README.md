# Grass Field 005

Puzzle matrix design studio for Granny's House Trials.

This subproject is a documentation-and-planning layer. There is no code here yet.
Its purpose is to consolidate the puzzle matrix vocabulary, expand it with design
analysis and lessons, and produce a complete design reference before matrix
concepts are wired as scenario action anchors into the sim in a future subproject.

## What is in here

- `PLAN.md` — scope and goals for this design work
- `docs/` — full consolidated puzzle matrix design documents
- `lessons/` — how-to lessons for reading, extending, and applying the matrix

## Relationship to existing work

The authored Granny's House map lives in `subprojects/grass-field-004`. That
subproject is a visual-only pass — no scenario anchors or round mechanics are
wired yet. A future subproject will connect the matrix vocabulary to actual
player actions in that map. This subproject defines what those actions should
mean and why.

The matrix design vocabulary originates from two earlier doc bundles:

- `subprojects/grass-field-004/grannys_house_trials_codex_docs/docs/design/`
- `subprojects/grass-field-004/grannys_house_worldbuilding_bundle/docs/`

This subproject consolidates, expands, and teaches those ideas.
