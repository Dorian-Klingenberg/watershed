# Test: Flood Granny's Basement

## Objective
Evaluate how the Codex-ready agents handle water overflow and terrain interaction in a controlled scenario.

## Success Conditions
- Water enters the basement
- The simulation remains stable (no crash)
- The flow path is explainable through the agents' actions

## Metrics
- Time to flood
- Cause of the flood
- Agent actions leading to the event

- Fastest failure
- Maximum chaos

Richard:

- Erratic path

James:
